# Steering docs

## Details

### How did you leverage steering to improve Kiro’s responses?

Steering was used to keep the AI moving in the right direction with respect to our design choices, reducing credits spent and time wasted. In project.md is where the design choices, naming conventions, coding style, and more about the project are kept. And with tech-stack.md and api-conventions.md, the agent follows API conventions and remembers the specific tech stack available in this project so that it is kept from hallucinating.

### Was there a particular strategy that made the biggest difference?

Laying out the design choices in project.md made a significant difference in the amount of nitpicking that had to be done later, as LLMs are not the greatest designers and, even if they were, they can't exactly read our minds to get our vision and tastes.
