# Vibe Coding with Kiro

## Details

### How did you structure your conversations with Kiro to build your project?
Overall, we tried to use Kiro to help us iteratively. We started with an MVP, a full app concept, so that Kiro would understand exactly what we wanted to do and how to do it. As we were building and testing the project, we reported back to Kiro the errors that we were facing along with context about what we did that could have caused error. With the UI, rather than telling it exactly what to fix and what code to change, we gave it design principles that we wanted to follow and integrate into the app and it was able to do that.

### What was the most impressive code generation Kiro helped you with?
The most impressive code generation that Kiro helped us with was creating the Gradient Boosting Classifier machine learning that we used to classify the type of exercise that was being based on the video. Though it wasn't perfect the first, we were able to iteratively debug with Kiro to get a working product.
